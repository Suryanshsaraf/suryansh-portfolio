## 🚀 Key Features

* **Interactive 3D Background:** A particle system powered by Three.js that reacts dynamically to movement.
* **Cursor Spotlight & Micro-Animations:** Fluid mouse-following hover effects and interactive button states.
* **Kinetic Smooth Scroll:** Powered by Lenis for seamless browsing across sections.

## 🛠️ Quick Start (Local Run)

If you'd like to run the website locally on your computer, you can do so by following these simple steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open the site:**
   Open your browser and navigate to the local address displayed in your terminal (usually `http://localhost:5173`).
